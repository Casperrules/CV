Audiobook Desktop Application

This app was developed using Python, I used the google text to speech pretrained model gtts module for python to make the actual conversion of text material to audio. I also used the playsound module in python to actually read the book after conversion if the user wanted to. PyPDF2 module was used to convert the pdf files to text. The user was asked to choose the pdf file they want to convert to audiobook and after conversion, the user can also specify where to store the audiobook. The user also was given an option to play the converted audiobook after the conversion.

The application asked the user to enter required information and returned the predicted fare. The whole project was done using Python.
